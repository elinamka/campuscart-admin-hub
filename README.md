# CampusCart Admin Hub

## Scheduled meals, delivered right on time.

An investor-ready SaaS operations dashboard for managing CampusCart – a meal scheduling and delivery service designed for busy university students.

### 🎯 Overview

CampusCart Admin Hub is the internal management system used by CampusCart staff to oversee operations, payments, schedules, customers, analytics, and deliveries. The interface is visually comparable to Stripe Dashboard, Notion, Linear, and Uber Eats Merchant Dashboard.

### ✨ Features

- **Dashboard Overview** - KPI cards with live visual indicators
- **Order Management** - Advanced order table with filtering, sorting, and bulk actions
- **Customer Management** - Customer database with profiles and history
- **Vendor Management** - Vendor database and operations
- **Delivery Queue** - Simple operational queue for single-rider model
- **Delivery Scheduler** - Professional scheduling (Daily/Weekly/Monthly views)
- **Payment Tracking** - Mobile Money & Paystack integration
- **Analytics & Reporting** - Beautiful charts and data visualization
- **Live Activity Feed** - Real-time order and system updates
- **Automation Center** - Workflow automation dashboard
- **Notifications Center** - System alerts and notifications
- **Settings** - Business profile, payment, security settings


### 📊 Operating Model

CampusCart operates with:
- **ONE rider only** (no complex rider assignment systems)
- **Manual vendor communication**
- **Simple delivery queue** for operational management
